# 11 — App Store Metadata and Screenshots

Status: Pending

## Goal

Prepare all customer-facing App Store Connect content.

## App information

- [ ] Confirm the final app name.
- [ ] Write subtitle.
- [ ] Write full description.
- [ ] Choose accurate keywords.
- [ ] Choose primary and optional secondary categories.
- [ ] Add copyright.
- [ ] Add Support URL.
- [ ] Add Privacy Policy URL.
- [ ] Add Marketing URL if desired.
- [ ] Prepare version release notes.

## Screenshots

- [ ] Capture required 6.9-inch iPhone screenshots.
- [ ] Use fictional/demo account information.
- [ ] Show actual app use, not only login or splash screens.
- [ ] Include pantry inventory.
- [ ] Include shopping cart.
- [ ] Include barcode/expiration scanning.
- [ ] Include shared pantry or reminders if advertised.
- [ ] Ensure overlays and claims exactly match available functionality.

## Product configuration

- [ ] Select free/paid price.
- [ ] Select distribution territories.
- [ ] Select release method.
- [ ] Verify the app name/SKU/App Store record.

## Done when

Every required metadata field is complete and every screenshot accurately
represents the submitted binary.
